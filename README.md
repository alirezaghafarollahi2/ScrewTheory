# ScrewTheory
Code for screw strength prediction in BCC random alloys based on Maresca-Curtin-Ghafarollahi theory

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/alirezaghafarollahi2/ScrewTheory/blob/main/strength_prediction.ipynb)

## Files

- `strength_prediction.ipynb` — interactive notebook for running the model
- `strength_prediction.py` — Python module with the core functions

## How to use

1. Click the Colab badge above.
2. Run the setup cell.
3. Enter the required material parameters.
4. Run the notebook to generate predictions.
5. Save the CSV and PNG outputs.

## Notes

- GitHub shows notebooks as static files.
- Use Google Colab to run the notebook interactively.
